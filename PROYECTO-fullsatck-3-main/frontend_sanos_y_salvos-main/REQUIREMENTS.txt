## ================================================================
## PROCEDIMIENTO DE INSTALACIÓN
## ================================================================

### Paso 1: Requisitos del Sistema
1. Instalar Node.js desde https://nodejs.org/
   - Descarga LTS (Long Term Support)
   - Mínimo v18.0.0

2. Verificar instalación:
   node --version    # Debe mostrar v18.x.x o superior
   npm --version     # Debe mostrar 9.x.x o superior

### Paso 2: Clonar/Acceder al Proyecto
cd "ruta/al/proyecto"

### Paso 3: Ejecutar Setup

#### Windows (Recomendado):
.\setup.ps1

Si recibes error de ejecución, ejecuta:
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser

#### Mac/Linux (Recomendado):
bash setup.sh

#### Manual (Todos los sistemas):
# Instalar backend
cd backend
npm install
cd ..

# Instalar frontend
npm install

# Crear archivos .env
copy .env.example .env
copy backend\.env.example backend\.env

### Paso 4: Verificar Instalación
# Health check del backend
curl http://localhost:5000/api/health

## ================================================================
## RESOLUCIÓN DE PROBLEMAS
## ================================================================

### Error: "npm: comando no encontrado"
Solución: Node.js no está instalado correctamente
- Descarga desde https://nodejs.org/
- Reinicia tu terminal después de instalar

### Error: "puerto ya en uso"
Solución: Otro proceso está usando el puerto
- Frontend (5173): Busca y cierra otro Vite
- Backend (5000): Busca y cierra otro Express

### Error: "módulo no encontrado"
Solución: Ejecuta npm install en la carpeta correspondiente
- Backend: cd backend && npm install && cd ..
- Frontend: npm install

### Error: CORS en frontend
Solución: Asegúrate de que:
1. Backend está corriendo en puerto 5000
2. VITE_API_URL en .env apunta a http://localhost:5000/api
3. Backend tiene CORS habilitado
